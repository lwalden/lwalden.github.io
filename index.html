Poker Hand Evaluator

This is a C# Console App. I wrote this while learning some basic C# syntax. Along with basics like enums, classes, switches, etc, this app showcases my ability to write logical algorithms to solve problems.

The repo for this app contains an .exe of the app, should you wish to run it without building the solution in Visual Studios or another IDE.

Current functionality includes: draw a card (up to 7), display your hand, and display the best possible 5 (or fewer) card hand you can make with your cards. Future updates will add more functionality such as discarding a card, community cards, and evaluating a hand's strength given a number of opponents and community cards.
